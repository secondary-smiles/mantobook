telnetd

NAME

telnetd − Telnet server

SYNOPSIS

telnetd [OPTION...]

DESCRIPTION

DARPA telnet protocol server
−D, −−debug[=LEVEL]

set debugging level

−E, −−exec−login=STRING

set program to be executed instead of /usr/bin/login

−h, −−no−hostinfo

do not print host information before login has been completed

−l, −−linemode[=MODE]

set line mode

−n, −−no−keepalive

disable TCP keep−alives

−U, −−reverse−lookup

refuse connections from addresses that cannot be mapped back into a symbolic name

−?, −−help

give this help list

−−usage

give a short usage message

−V, −−version

print program version

Mandatory or optional arguments to long options are also mandatory or optional for any corresponding short options.
