asn1_dup_node

NAME

asn1_dup_node − API function

SYNOPSIS

#include <libtasn1.h>

asn1_node asn1_dup_node(asn1_node_const src, const char * src_name);

ARGUMENTS

asn1_node_const src

Source asn1 node.

const char * src_name

Field name in source node.

DESCRIPTION

Create a deep copy of a asn1_node variable. This function will return an exact copy of the provided structure.

RETURNS

Return NULL on failure.
