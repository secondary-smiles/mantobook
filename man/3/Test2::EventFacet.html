Test2::EventFacet

NAME

Test2::EventFacet − Base class for all event facets.

DESCRIPTION

Base class for all event facets.

METHODS

$key = $facet_class−>facet_key()

This will return the key for the facet in the facet data hash.

$bool = $facet_class−>is_list()

This will return true if the facet should be in a list instead of a single item.

$clone = $facet−>clone()
$clone = $facet−>clone(%replace)

This will make a shallow clone of the facet. You may specify fields to override as arguments.
