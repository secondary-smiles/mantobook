BN_swap

NAME

BN_swap − exchange BIGNUMs

SYNOPSIS

#include <openssl/bn.h>
void BN_swap(BIGNUM *a, BIGNUM *b);

DESCRIPTION

BN_swap() exchanges the values of a and b.

RETURN VALUES

BN_swap() does not return a value.
