One-Call function for TPM2_PCR_SetAuthValue

This function invokes the TPM2_PCR_SetAuthValue command in a one-call variant. This means the function will block until the TPM response is available. All input parameters are const. The memory for non-simple output parameters is allocated by the function implementation.

pcrHandle Handle for a PCR that may have an authorization value set.

shandle2 Second session handle.

auth The desired authorization value.
