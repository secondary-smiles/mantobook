gnutls_x509_crt_equals2

NAME

gnutls_x509_crt_equals2 − This function compares a gnutls_x509_crt_t cert with DER data

SYNOPSIS

#include <gnutls/x509.h>

unsigned gnutls_x509_crt_equals2(gnutls_x509_crt_t cert1, const gnutls_datum_t * der);

ARGUMENTS

gnutls_x509_crt_t cert1

The first certificate

const gnutls_datum_t * der

A DER encoded certificate

DESCRIPTION

This function will compare an X.509 certificate structures, with DER encoded certificate data.

RETURNS

On equality non−zero is returned, otherwise zero.

SINCE

3.5.0
