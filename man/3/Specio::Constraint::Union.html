Specio::Constraint::Union

NAME

Specio::Constraint::Union − A class for union constraints

VERSION

version 0.48

SYNOPSIS

my $type = Specio::Constraint::Untion−>new(...);

DESCRIPTION

This is a specialized type constraint class for unions, which will allow a value which matches any one of several distinct types.

API

This class provides all of the same methods as Specio::Constraint::Simple, with a few differences:

Specio::Constraint::Union−>new( ... )
The "parent" parameter is ignored if it passed, as it is always "undef"

The "inline_generator" and "constraint" parameters are also ignored. This class provides its own default inline generator subroutine reference.

Finally, this class requires an additional parameter, "of". This must be an arrayref of type objects.

$union−>of
Returns an array reference of the individual types which makes up this union.

ROLES

This class does the Specio::Constraint::Role::Interface and Specio::Role::Inlinable roles.
