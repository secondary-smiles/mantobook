TAP::Parser::Result::Comment

NAME

TAP::Parser::Result::Comment − Comment result token.

VERSION

Version 3.44

DESCRIPTION

This is a subclass of TAP::Parser::Result. A token of this class will be returned if a comment line is encountered.

1..1
ok 1 − woo hooo!
# this is a comment
