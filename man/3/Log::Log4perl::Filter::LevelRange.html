Log::Log4perl::Filter::LevelRange

NAME

Log::Log4perl::Filter::LevelRange − Filter for a range of log levels

SYNOPSIS

log4perl.filter.Match1 = Log::Log4perl::Filter::LevelRange
log4perl.filter.Match1.LevelMin = INFO
log4perl.filter.Match1.LevelMax = ERROR
log4perl.filter.Match1.AcceptOnMatch = true

DESCRIPTION

This Log4perl custom filter checks if the current message has a priority matching a predefined range. The "LevelMin" and "LevelMax" parameters define the levels (choose from "DEBUG", "INFO", "WARN", "ERROR", "FATAL") marking the window of allowed messages priorities.

"LevelMin" defaults to "DEBUG", and "LevelMax" to "FATAL".

The additional parameter "AcceptOnMatch" defines if the filter is supposed to pass or block the message ("true" or "false").
