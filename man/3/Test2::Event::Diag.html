Test2::Event::Diag

NAME

Test2::Event::Diag − Diag event type

DESCRIPTION

Diagnostics messages, typically rendered to STDERR.

SYNOPSIS

use Test2::API qw/context/;
use Test2::Event::Diag;
my $ctx = context();
my $event = $ctx−>diag($message);

ACCESSORS

$diag−>message

The message for the diag.
