Alien::Base::FAQ

NAME

Alien::Base::FAQ − Frequently asked questions

VERSION

version 2.80

SYNOPSIS

% perldoc Alien::Build::Manual::FAQ
% perldoc Alien::Base::ModuleBuild::FAQ

DESCRIPTION

This used to answer FAQs regarding the only way to author an Alien::Build distribution, which was with Alien::Base::ModuleBuild. You should now seriously consider using the newer more reliable method which is via Alien::Build and alienfile.
