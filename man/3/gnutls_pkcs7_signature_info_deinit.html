gnutls_pkcs7_signature_info_deinit

NAME

gnutls_pkcs7_signature_info_deinit − API function

SYNOPSIS

#include <gnutls/pkcs7.h>

void gnutls_pkcs7_signature_info_deinit(gnutls_pkcs7_signature_info_st * info);

ARGUMENTS

gnutls_pkcs7_signature_info_st * info

should point to a gnutls_pkcs7_signature_info_st structure

DESCRIPTION

This function will deinitialize any allocated value in the provided gnutls_pkcs7_signature_info_st.

SINCE

3.4.2
