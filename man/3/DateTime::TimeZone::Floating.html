DateTime::TimeZone::Floating

NAME

DateTime::TimeZone::Floating − A time zone that is always local

VERSION

version 2.60

SYNOPSIS

my $floating_tz = DateTime::TimeZone::Floating−>new;

DESCRIPTION

This class is used to provide the DateTime::TimeZone API needed by DateTime.pm, but for floating times, as defined by the RFC 2445 spec. A floating time has no time zone, and has an effective offset of zero.

USAGE

This class has the same methods as a real time zone object. The "short_name_for_datetime()" method returns the string "floating" and the "category()" method returns "undef".
