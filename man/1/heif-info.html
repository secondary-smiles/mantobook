heif-info

NAME

heif-info − show information on HEIC/HEIF file

SYNOPSIS

heif-info [−d|--dump-boxes] [−h|--help] filename

DESCRIPTION

heif-info Show information on HEIC/HEIF file.

OPTIONS

−d, −−dump-boxes

Show a low-level dump of all MP4 file boxes.

−help, −−help

Show help.

EXIT STATUS

0

Success

1

Failure (syntax or usage error; error while loading image).
